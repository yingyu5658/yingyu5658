# Hello, I'm Verdant

I'm a developer who enjoys working across the stack — from system-level tinkering with **Linux** and **C**, to building efficient backends with **Go**, and creating interactive experiences with **JavaScript**. My editor of choice is **Emacs**, where I try to craft my perfect development environment.

I believe in building simple, effective tools and sharing knowledge. My blog are where I document my learning and occasional thoughts.
